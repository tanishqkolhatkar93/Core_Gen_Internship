# 🎯 Real-Time Object Detection System (YOLO / MobileNet-SSD)

## 🚀 Project Overview
This project implements a real-time object detection system using YOLO and MobileNet-SSD with OpenCV.

---

## 🔍 Key Features
- Real-time object detection using webcam  
- Image and video detection  
- Pre-trained models (YOLO / SSD)  
- Object labeling with confidence scores  
- Fast inference  

---

## 🧠 Working Principle
Input → Model → Detection → Labels → Output

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- NumPy  
- YOLO / MobileNet-SSD
